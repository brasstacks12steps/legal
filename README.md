# legal
Data Deletion & Account Management
App name: Brass Tacks 12 Steps
Brass Tacks 12 Steps gives users full control over their data directly within the app.

How to delete specific data (without deleting your account)
Users can delete individual types of data at any time by following these steps inside the app:
Open the app and go to Settings
Select Data Management
Choose one of the available options:
Delete Daily Reflections
Delete Daily Inventories
Delete All Step Work
Confirm deletion when prompted
Deleted data is permanently removed and cannot be recovered.

How to export your data
Users can download a copy of their data at any time:
Open Settings
Select Data Management
Tap Export All Data
A JSON file containing your data will be downloaded

How to delete your account
Account deletion is available inside the app:
Open Settings
Navigate to Account / Data Management
Select Delete Account
Confirm deletion
When an account is deleted, all associated user data is permanently removed.

Data retention
Deleted data is removed immediately
No deleted data is retained beyond necessary legal or operational requirements

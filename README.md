Data Deletion & Account Management
App name: Brass Tacks 12 Steps
Brass Tacks 12 Steps gives users full control over their data directly within the app.

How to delete your account and associated data
Users can delete their account directly within the app at any time:
Open the app
Navigate to Settings
Select Account / Data Management
Choose Delete Account
Confirm the deletion request
When an account is deleted:
The user account is permanently removed
All associated personal data is permanently deleted
Deleted data cannot be recovered
No additional action is required after confirming deletion in the app.

How to delete specific data without deleting your account
Your App Name allows users to delete specific types of data without deleting their entire account.
To delete individual data types:
Open the app
Go to Settings → Data Management
Select one of the following options:
Delete Daily Reflections
Delete Daily Inventories
Delete All Step Work
Confirm deletion when prompted
Only the selected data is deleted. The user account remains active.

How to export your data
Users can download a copy of their data at any time:
Open the app
Go to Settings → Data Management
Select Export All Data
A JSON file containing the user’s data will be generated and downloaded.

Data retention and processing
Deleted data is permanently removed from our systems
No deleted data is retained beyond necessary operational or legal requirements
Exported data is provided directly to the user and is not stored by us
